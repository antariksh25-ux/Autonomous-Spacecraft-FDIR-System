# FDIR Showcase Website

Modern Next.js web application showcasing the Autonomous FDIR system.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Features

- **Live Telemetry Simulation**: Real-time spacecraft telemetry visualization
- **Anomaly Detection Demo**: Interactive fault detection with visual feedback
- **Recovery Actions**: Automated recovery recommendations
- **Performance Metrics**: Real-time evaluation metrics
- **Simulation Controls**: Inject different anomaly types

## Technology Stack

- **Next.js 14** - React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling
- **Framer Motion** - Smooth animations
- **Recharts** - Telemetry visualization
- **Lucide Icons** - Modern icon set

## Components

- `Hero.tsx` - Landing section
- `TelemetryDashboard.tsx` - Live telemetry charts
- `FaultPanel.tsx` - Anomaly visualization
- `RecoveryFlow.tsx` - Recovery action display
- `Architecture.tsx` - System architecture diagram
- `Metrics.tsx` - Performance metrics
- `Controls.tsx` - Simulation controls

## Build

```bash
npm run build
npm start
```

## Deploy

Deploy easily on Vercel, Netlify, or any platform supporting Next.js.
